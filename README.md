# How to flash
1. Install HyperOS 1.0 and flash it in your phone (1.0.8.0.UNQCNXM Recommended) <br>
2. Extract Port file in Platform tools <br>
3. Run flash file <br>

# Note:
1. Don't use Always On Display(or experience bootloop)
2. Be patient during first time setup
3. Use device for 1-2 days for better stability
